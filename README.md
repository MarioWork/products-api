# Products API


### Product
-   [x] /product/list - get - returns all products
-   [ ] /product/id - get - returns a single product
-   [ ] /product/id - delete - removes a product
-   [ ] /product/id - patch - updates a product
-   [x] /product/ - post - creates a product
